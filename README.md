# Almeida Monorepo

This repository is organized as a monorepo containing the following applications:

- **`apps/front-end`**: The main institutional website (Vite + React + Shadcn UI).
- **`apps/back-office`**: Administration dashboard (Vite + React).
- **`apps/back-end`**: Backend services (NestJS).

## Commands

From the root directory, you can manage all applications using `pnpm`:

- `pnpm install`: Install dependencies for all apps.
- `pnpm dev`: Run all applications in development mode.
- `pnpm build`: Build all applications for production.
- `pnpm lint`: Run linting for all applications.
